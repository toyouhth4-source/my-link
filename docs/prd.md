## 1. 프로젝트 개요

*   **프로젝트명:** 마이링크 (MyLink) - 개인 및 지정 그룹용
*   **프로젝트 목적:** '나 자신(개발자/학생)'을 가장 잘 소개하고 증명할 수 있는 고품질 통합 프로필 사이트 구축.
*   **디자인 컨셉:** **네오 브루탈리즘 (Neo-Brutalism)** 
*   **열람 정책:** 일반 브라우저 접속은 허용하되, 검색 엔진 노출은 차단.

---

## 2. 핵심 기능 목록 (개발자 포트폴리오 중심)

### 2.1 필수 기능 (Required)
*   **[P1] '나'를 소개하는 핵심 프로필 뷰** (사진, 전공, 학번 등 개인 아이덴티티 부각)
*   **[P1] 아카이브 보드 및 다중 링크** (프로젝트 갤러리 형태로 Github, 블로그 등 연결)
*   **[P1] 네오 브루탈리즘 테마 전면 적용** (UI/UX 뼈대 전체에 굵은 선형 디자인 반영)
*   **[P1] 구글 소셜 로그인 기반의 폐쇄형 간편 방명록** (`아이디 (실명)` 형태 표기)
*   **[P2] Google API 중심 확장 기능** (Drive 연동 이력서 다운로드, Analytics 방문 통계)

### 2.2 심화 및 고급 UX 기능 (Advanced)
*   **[P3] Github Contribution(잔디) 시각화 위젯** (외부 API 활용, 나의 개발 성실도 시각 증명)
*   **[P3] 키보드 단축키(Cmd/Ctrl + K) 기반 커맨드 팔레트** (마우스 없이 사이트 내비게이션 가능)
*   **[P3] 기술 스택 무빙 텍스트(Marquee) 애니메이션** (화면 어딘가에서 기술 스택이 전광판처럼 흐르는 디자인 풀이)
*   **[P4] 프라이빗 컨셉을 살린 위트 있는 404 및 보안 접근 거부 페이지**
*   **[P4] 소소한 마이크로 인터랙션** (버튼 호버 애니메이션, 클릭 팝 등)

---

## 3. 기능 상세 설명 및 UI/UX 포커스

### 3.1 인터랙티브 네오 브루탈리즘과 모션
*   눈에 띄는 고대비 색상을 바탕으로, **무빙 텍스트(Marquee)**나 단축키 인터페이스 등 개발자기 선호할법한 긱(Geek)한 에너지를 발산하도록 디자인합니다.

### 3.2 신뢰감을 주는 '개발자 증명' 컴포넌트
*   추가적인 클릭이나 말글 없이도 "이 학생은 매일 코딩합니다"를 증명할 수 있도록 **Github의 잔디를 긁어와 네오 브루탈리즘 톤으로 변환**하여 노출함으로써 평가자에게 강력한 신뢰를 줍니다.
*   `Ctrl+K` 검색창을 열 수 있다는 것 자체가 "이 학생은 개발 편의성과 최신 트렌드를 매우 잘 이해하고 있구만" 이라는 긍정적 후광 효과를 낳습니다.
